Как получить ключи Yandex Cloud

Шаг 1. Создайте сервисный аккаунт

  1. Зайдите в Yandex Cloud Console.
  2. Выберите каталог (Folder), где будет работать сервис.
  3. Откройте раздел \"Сервисные аккаунты\" и создайте новый.

Шаг 2. Назначьте роли

Сервисному аккаунту нужны роли:

  • ai.speechkit-stt.user — для транскрибации.
  • ai.languageModels.user — для использования YandexGPT.
  • Дополнительно: storage.viewer (если планируется использование Object Storage).

Шаг 3. Создайте API-ключ

  1. В разделе сервисного аккаунта выберите вкладку \"API-ключи\".
  2. Создайте ключ и сохраните его — он будет нужен для полей YANDEX_API_KEY и (при необходимости) YANDEXGPT_API_KEY.
  3. Скопируйте Folder ID из верхнего меню консоли или раздела каталога.

Советы по безопасности

  • Не делитесь API-ключами с третьими лицами.
  • При потере ключа немедленно отозовите его в консоли Yandex Cloud.
  • Храните ключи в защищённом месте, а в приложении используйте поле \"Очистить\" при работе на публичных компьютерах.
